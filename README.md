## 📌 项目简介
3Dmax测量脚本，悬浮窗口显示两点间的距离，XYZ增量结果，模型两点间的尺寸标注
---

## 🛠️ 使用说明
- 1，拖进去3Dmax窗口即可生效
- 2，本人使用的是3Dmax2021版本，其他版本未做测试
- 3，在自定义-工具栏My-Tools-高级两点则距(带3D文字标记)/两点测距（带橡皮筋辅助线）
- 4，在热键里面能赋予快捷键，名字为：高级两点则距/两点测距（高级两点则距(带3D文字标记)/两点测距（带橡皮筋辅助线））
